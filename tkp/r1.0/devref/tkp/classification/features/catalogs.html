tkp.classification.features.catalogs – Catalog source matching¶

Module that checks the database for source associations

If the database is not available or the database module cannot be imported, functions will silently return None.

tkp.classification.features.catalogs.match_catalogs(transient)[source]¶

Match transient source with nearest catalog source

Iterate through available catalogs, and return the nearest source for each catalog. Each such source is a dictionary with keys catsrcid, catsrcname, catid, catname, ra, decl, ra_err, decl_err, dist_arcsec, assoc_r. See match_nearests_in_catalogs() for details.

The returned dictionary contains the catalog name as its key, and a source as the corresponding value.
