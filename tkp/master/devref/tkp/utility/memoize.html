tkp.utility.memoize – Memoization decorator¶

class tkp.utility.memoize.Memoize(funct)[source]¶

Decorator to cache the results of methods.

Examples in e.g. image.py:

@Memoize
def _grids(self):
    return self.__grids()
grids = property(fget=_grids, fdel=_grids.delete)
