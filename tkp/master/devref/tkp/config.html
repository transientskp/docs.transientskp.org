Configuration¶

tkp.config.database_config(pipe_config=None, apply=False)[source]¶

sets up a database configuration using the settings defined in a pipeline.cfg (if supplied) and optionally overriden by the environment. The following environment variables are recognized:

* TKP_DBENGINE
* TKP_DBNAME
* TKP_DBUSER
* TKP_DBPASS
* TKP_DBHOST
* TKP_DBPORT
Parameters:
  • pipe_config – a ConfigParser object
  • apply – apply settings (configure db connection) or not
