Feature extraction package¶

Routines from tkp.classification.features

The features subpackage contains a few modules, one for each type of characteristics. At the moment, the two modules below are the only ones available:

  • tkp.classification.features.lightcurve – Lightcurve feature extraction utilities
  • tkp.classification.features.catalogs – Catalog source matching
