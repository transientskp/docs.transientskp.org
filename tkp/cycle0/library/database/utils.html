tkp.database.utils – Database utilities¶

The back end of our database ORM. This is where all the SQL queries hide.

This sub-package was initially one giant module, but it became unwieldy. Hence we simply dump everything into the sub-package namespace, because it makes the client code agnostic about the layout of the actual modules.
